<objective>
## Goal
Replace the flat `ResultList` in `PrefixesTab` with a `PreviewTable` widget — a `Gtk.ListStore`-based view with a checkbox column, "Current" and "Proposed" columns showing the before/after correction, and Select All / Deselect All buttons. Enrich `detect_issues()` to return the correction data needed to populate these columns.

## Purpose
Plan 05-01 delivered detection. The ROADMAP deliverable for Phase 5 is "preview-before-commit workflow". The preview table is the UX layer that lets the user see exactly what will change and choose which rows to apply, before Plan 05-03 writes anything to the GRAMPS database.

## Output
- `detect_issues()` enriched with `prefix_found`, `field_type`, `current_value`, `proposed_value`
- New `PreviewTable` class in `tab_prefixes.py` — Gtk.ListStore with checkbox, Name, ID, Field, Current, Proposed columns
- `PrefixesTab` updated to use `PreviewTable`, with Select All / Deselect All toolbar buttons
- `from widgets import ResultList` import removed (no longer needed)
</objective>

    **1b. Add `PreviewTable` class**

    Insert immediately above the `# Tab widget` section separator (after `detect_issues`, before `class PrefixesTab`).

    **ListStore column indices (class constants):**
    | Index | Name | Type | Purpose |
    |-------|------|------|---------|
    | 0 | _COL_CHECKED | bool | Row selected for apply |
    | 1 | _COL_NAME | str | Person display name |
    | 2 | _COL_ID | str | GRAMPS ID |
    | 3 | _COL_FIELD | str | "Given name" or "Surname" |
    | 4 | _COL_CURRENT | str | Field value with prefix e.g. "van Jan" |
    | 5 | _COL_PROPOSED | str | e.g. "'Jan'  •  prefix: van" |
    | 6 | _COL_HANDLE | str | Person handle (hidden) |
    | 7 | _COL_PREFIX | str | Matched prefix string (hidden, for apply) |
    | 8 | _COL_FTYPE | str | "given"/"surname" (hidden, for apply) |

    **`__init__(self, on_activate=None)`:**
    - `Gtk.Box.__init__(self, orientation=Gtk.Orientation.VERTICAL, spacing=4)`
    - `self._on_activate = on_activate`
    - `self.store = Gtk.ListStore(bool, str, str, str, str, str, str, str, str)`
    - `self.tree = Gtk.TreeView(model=self.store)`
    - `self.tree.set_enable_search(False)`
    - `self.tree.connect("row-activated", self._row_activated)`

    **Columns to append:**

    Column 0 — "☑" (checkbox):
    - `renderer = Gtk.CellRendererToggle()`
    - `renderer.connect("toggled", self._on_toggled)`
    - `col = Gtk.TreeViewColumn("☑", renderer, active=self._COL_CHECKED)`
    - `col.set_min_width(30)`; `col.set_resizable(False)`

    Column 1 — "Name":
    - Standard `Gtk.CellRendererText`, `text=self._COL_NAME`, min_width=160, resizable

    Column 2 — "ID":
    - Standard `Gtk.CellRendererText`, `text=self._COL_ID`, min_width=70, resizable

    Column 3 — "Field":
    - Standard `Gtk.CellRendererText`, `text=self._COL_FIELD`, min_width=90, resizable

    Column 4 — "Current":
    - Standard `Gtk.CellRendererText`, `text=self._COL_CURRENT`, min_width=140, resizable

    Column 5 — "Proposed":
    - Standard `Gtk.CellRendererText`, `text=self._COL_PROPOSED`, min_width=200, resizable

    **Layout (vertical Gtk.Box):**
    - Scrolled window (auto/auto, ShadowType.IN, expand=True) containing TreeView
    - Status label (halign=START, margin_start=4, margin_bottom=4, expand=False)
    - `self.status_label = Gtk.Label(label="Ready.")`

    **Public methods:**

    `clear(self)`:
    - `self.store.clear()`
    - `self.status_label.set_text("Ready.")`

    `set_status(self, text)`:
    - `self.status_label.set_text(text)`

    `append_row(self, checked, name, gramps_id, field, current, proposed, handle, prefix, ftype)`:
    - `if not name: return`  — guard for empty sentinel
    - `self.store.append([checked, name, gramps_id, field, current, proposed, handle, prefix, ftype])`

    `get_selected_findings(self)`:
    - Iterate over `self.store`; for each row where `row[self._COL_CHECKED]` is True:
      - Append `{"handle": row[self._COL_HANDLE], "prefix_found": row[self._COL_PREFIX], "field_type": row[self._COL_FTYPE], "proposed_value": row[self._COL_CURRENT][len(row[self._COL_PREFIX]) + 1:]}`
    - Return the list
    - Note: `proposed_value` is recomputed from current and prefix — no need to store separately

    **Private methods:**

    `_on_toggled(self, renderer, path)`:
    - `it = self.store.get_iter(Gtk.TreePath(path))`
    - `self.store[it][self._COL_CHECKED] = not self.store[it][self._COL_CHECKED]`

    `_row_activated(self, tree, path, column)`:
    - `it = self.store.get_iter(path)`
    - `handle = self.store.get_value(it, self._COL_HANDLE)`
    - `if handle and self._on_activate: self._on_activate(handle)`

    `select_all(self)`:
    - Iterate `self.store`; set `row[self._COL_CHECKED] = True` for every row

    `deselect_all(self)`:
    - Iterate `self.store`; set `row[self._COL_CHECKED] = False` for every row

    Avoid: calling GTK from background threads. `append_row` is called via `GLib.idle_add` — safe.
