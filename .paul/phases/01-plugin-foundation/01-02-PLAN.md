<objective>
## Goal
Replace the placeholder window with a tabbed Gtk.Notebook containing one tab per tool (Island Detection, Missing Data, Impossibilities, Name Prefixes), and create a reusable ResultList widget that all four tool tabs will use to display scan results.

## Purpose
The tabbed layout is the permanent UI skeleton — all subsequent phases (2–5) will populate one tab each. The ResultList widget is the shared infrastructure for displaying people/families found by each tool, making it the most reused component in the plugin.

## Output
- `grampsclean/widgets.py` — new module with `ResultList` widget (Gtk.TreeView-based)
- `grampsclean/tool.py` — updated: placeholder replaced with Gtk.Notebook, four tabs each containing a ResultList placeholder
</objective>

<task type="auto">
  <name>Task 1: Create ResultList widget module</name>
  <files>grampsclean/widgets.py</files>
  <action>
    Create grampsclean/widgets.py with a single class: `ResultList`.

    **ResultList(Gtk.Box):**
    - Constructor: `__init__(self, columns)`
      - `columns`: list of (column_title, column_id) tuples, e.g.
        [("Name", "name"), ("ID", "gramps_id"), ("Detail", "detail")]
      - Call `Gtk.Box.__init__(self, orientation=Gtk.Orientation.VERTICAL, spacing=4)`
      - Build a `Gtk.ListStore` with one `str` column per entry in `columns`
      - Build a `Gtk.TreeView` with the ListStore as model
      - For each column, add a `Gtk.TreeViewColumn` with a `Gtk.CellRendererText`
        using `add_attribute(renderer, "text", col_index)`
      - Set TreeView properties:
        - `set_headers_visible(True)`
        - `set_enable_search(False)`
        - `get_selection().set_mode(Gtk.SelectionMode.SINGLE)`
      - Wrap TreeView in a `Gtk.ScrolledWindow`:
        - `set_policy(Gtk.PolicyType.AUTOMATIC, Gtk.PolicyType.AUTOMATIC)`
        - `set_shadow_type(Gtk.ShadowType.IN)`
      - Add scrolled window with `pack_start(sw, True, True, 0)` (expands)
      - Add a `Gtk.Label` status bar below:
        - Default text: "Ready — click Scan to begin"
        - `set_halign(Gtk.Align.START)`
        - `set_margin_start(4)` and `set_margin_bottom(4)`
        - Store as `self.status_label`
        - Add with `pack_start(status_label, False, False, 0)`

    **Public methods:**
    - `clear(self)`: calls `self.store.clear()` and resets status to "Ready — click Scan to begin"
    - `set_status(self, text)`: sets `self.status_label.set_text(text)`
    - `append_row(self, *values)`: calls `self.store.append(list(values))`

    Store the ListStore as `self.store` and TreeView as `self.tree`.

    Use `gi.require_version("Gtk", "3.0")` at top.
    Avoid: importing anything from gramps — this widget must be GRAMPS-agnostic.
  </action>

    with a proper Gtk.Notebook containing four tabs.

    **Tab definitions** — define as a module-level constant:
    ```python
    TABS = [
        ("Island Detection",  [("Name", 0), ("Group Size", 1), ("Detail", 2)]),
        ("Missing Data",      [("Name", 0), ("ID", 1),         ("Missing", 2)]),
        ("Impossibilities",   [("Name", 0), ("ID", 1),         ("Issue", 2)]),
        ("Name Prefixes",     [("Name", 0), ("ID", 1),         ("Suggestion", 2)]),
    ]
    ```

    **_build_window changes:**
    - Create a `Gtk.Window()` as before
    - Create a `Gtk.Box(orientation=VERTICAL, spacing=0)` as the root container
    - Create a `Gtk.Notebook()`; set `set_tab_pos(Gtk.PositionType.TOP)`
    - For each (tab_label, columns) in TABS:
      - Create a `ResultList(columns)` instance
      - Add to notebook: `notebook.append_page(result_list, Gtk.Label(label=tab_label))`
      - Add padding around result_list: `result_list.set_border_width(8)`
    - Add notebook to root box with `pack_start(notebook, True, True, 0)`
    - Add a button bar below: `Gtk.Box(spacing=6)` with `set_border_width(8)`
      - Add a right-aligned "Close" button: `Gtk.Button(label="Close")`
        connected to `self.close`
      - Use `button_bar.pack_end(close_btn, False, False, 0)`
    - Add button bar to root box: `pack_start(button_bar, False, False, 0)`
    - Add root box to window: `window.add(root_box)`
    - Call `self.set_window(window, None, "GrampsClean \u2014 Data Quality Tools")`
    - Call `self.setup_configs("interface.grampsclean", 900, 600)`

    Store `self.notebook = notebook` for future phase access.

    Remove: the old `Gtk.Label` placeholder and old Close button wiring.
    Avoid: importing anything from gramps in widgets.py — keep it clean.
